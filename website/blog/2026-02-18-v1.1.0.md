**v1.1.0** brings TypeScript strict mode, comprehensive documentation, and a massively expanded test suite.

### TypeScript strict mode

The entire codebase now compiles with `strict: true` in `tsconfig.json`. This includes `strictNullChecks`, `strictPropertyInitialization`, `noImplicitAny`, and all other strict flags. This means better type safety and improved IDE support.

### Documentation

- Comprehensive JSDoc added to every source file
- New [Docusaurus documentation site](https://sui-js.siposdani87.com) with guides, API reference, and Algolia search
- Guides covering architecture, forms, HTTP requests, state management, and more

### Testing

The test suite grew from **180 to 1,187 tests** across 108 suites, bringing statement coverage from 52% to 75%.

### Upgrading

```bash
npm install @siposdani87/sui-js@1.1.0
```
