SUI-JS has reached **v1.0.0** — the first stable major release.

This milestone marks the framework as production-ready after years of iterative development from v0.2.0 through the v0.9.x series. The API surface is now considered stable.

### What's in v1.0.0

- Stable API for all 60+ UI components
- TypeScript-first codebase with full type declarations
- ESBuild-powered bundling (IIFE format)
- SCSS styling system with light/dark theme support

### Upgrading

```bash
npm install @siposdani87/sui-js@1.0.0
```

No breaking changes from v0.9.17 — this release stabilizes the existing API and fixes the `DateIO.parse()` method.
