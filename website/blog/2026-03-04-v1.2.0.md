**v1.2.0** continues the testing push and upgrades the tooling to the latest versions.

### Testing improvements

- **1,657 tests** across 108 suites (up from 1,187 in v1.1.0)
- Statement coverage increased to **88%** (branches 74%, functions 85%)
- Added `jest-axe` automated accessibility tests with ARIA attributes
- Proper `destroy()` methods for State, Screen, Page, and Form with listener cleanup
- Comprehensive tests for all menu modules

### Tooling upgrades

- ESLint 9 to 10 with flat config migration
- Stylelint 16 to 17
- Complete MDL-to-SUI migration with consistent field styling

### Upgrading

```bash
npm install @siposdani87/sui-js@1.2.0
```
