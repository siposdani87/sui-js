**v2.0.0** is a major release that modernizes the framework internals while keeping the public API largely compatible.

<!-- truncate -->

### Testing

- **2,212 tests** across 112 suites (up from 1,187 in v1.1.0)
- Statement coverage at **97%**, branch coverage at **88%**
- Added `jest-axe` automated accessibility tests with ARIA attributes
- Strict null checks and `noImplicitAny` enabled in test config

### Architecture

- Split `googleMap.ts` into focused modules (`mapMarkerOps`, `mapPolygonOps`, `mapLabel`)
- Split `operation.ts` into 7 focused utility modules
- Automatic DI via `static inject` for controllers and services

### Fetch migration

- `Xhr` module now uses the Fetch API instead of `XMLHttpRequest`
- `Promize` backed by native `Promise` internally
- Replaced `crypto-js` with native implementations

### Style modernization

- Dark mode consolidated via CSS custom properties (`--sui-*`)
- 42 separate Dark.scss files eliminated
- Indigo/orange palette, system fonts, normalized z-index scale
- WCAG AA color contrast compliance

### Tooling

- Conventional commits enforced via commitlint + husky
- Changelog automation with `conventional-changelog`
- TypeScript incremental builds, bundle size CI reporting
- Node.js engine requirement bumped to >=24.0.0

### Upgrading

```bash
npm install @siposdani87/sui-js@2.0.0
```
